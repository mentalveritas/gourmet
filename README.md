# 맛집 Review 웹페이지 템플릿

## 소개
이미지를 포함한 게시물 작성 및 댓글 기능이 담긴 커뮤니티 구축 웹 템플릿

## 개발환경
- Ubuntu 16.04 LTS
- Ruby : 2.3.1
- Rails : 5.0.0
- Database : Dev-> sqlite3, Prod -> postgresql

## 기능
- 게시물 CRUD
- 댓글 CRUD
- 회원가입, 로그인, 회원페이지
- 게시물/댓글 권한 관리
- 이미지 업로드 기능

## 추가예정
- 소셜 로그인 기능
- 회원가입시 이메일 인증 시스템
- 기본 Front-end 디자인 업데이트
